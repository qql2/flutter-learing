# Flutter 学习项目

这是一个用于学习Flutter的代码仓库，包含各种Flutter开发实践和示例。

## 项目目标
- 学习Flutter框架基础知识
- 掌握Flutter UI开发
- 理解Flutter状态管理
- 实践Flutter最佳开发模式

## 环境配置
1. Flutter SDK 要求:
   - Flutter 3.0.0 或更高版本
   - Dart 2.17.0 或更高版本

2. 开发工具:
   - VS Code 或 Android Studio
   - Flutter 和 Dart 插件
